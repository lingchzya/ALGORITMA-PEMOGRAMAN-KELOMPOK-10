# ALGORITMA-PEMOGRAMAN-KELOMPOK-10
Program Bangun Ruang (Java)
Deskripsi

Program ini dibuat menggunakan bahasa Java untuk menghitung volume beberapa bangun ruang, yaitu:

Kubus

Balok

Tabung

Pengguna dapat memilih bangun ruang yang ingin dihitung melalui menu di terminal, kemudian memasukkan nilai yang diperlukan seperti sisi, panjang, lebar, tinggi, atau jari-jari.

Program akan menampilkan hasil volume dari bangun ruang tersebut.
